# 获取支持

请根据问题类型选择入口：

- 可复现的软件错误：使用 [Bug 报告](https://github.com/fanshouheng/makeup-match-prototype/issues/new?template=bug_report.yml)。
- 新功能、使用场景和产品建议：前往 [Discussions](https://github.com/fanshouheng/makeup-match-prototype/discussions)。
- 准备提交代码：先阅读 [参与贡献](CONTRIBUTING.md)。
- 安全漏洞：按照 [安全政策](SECURITY.md) 私下报告。
- 博主申请、撤回、下架和删除：使用正式服务入口或正式部署页面公布的隐私联系邮箱。

请勿通过 GitHub 上传真人照片、面部特征、邮箱、授权证明或生产环境数据。GitHub 社区不受理博主身份核验，也不会通过 Issue 完成入库。
