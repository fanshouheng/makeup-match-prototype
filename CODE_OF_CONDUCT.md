# 社区行为准则

“妆容参照”希望建立一个尊重用户、尊重博主授权，也尊重不同经验水平的协作社区。

## 我们鼓励

- 围绕问题和证据讨论，不攻击个人动机或能力；
- 对新参与者说明背景、复现方法和可执行的下一步；
- 尊重不同肤色、性别表达、年龄、外貌特征和文化背景；
- 对涉及肖像、面部数据和个人信息的建议保持克制；
- 发现错误时及时更正，并接受维护者基于产品边界作出的决定。

## 不可接受的行为

- 侮辱、骚扰、歧视、威胁、跟踪或公开他人私人信息；
- 对用户或博主的外貌作贬损、排名或未经请求的评价；
- 未经授权上传、传播或分析真人照片、面部数据和联系方式；
- 冒充博主、伪造授权或利用项目进行身份识别；
- 刷屏、恶意引战、持续偏离主题或干扰维护工作。

## 适用范围与处理

本准则适用于仓库的 Issue、Discussion、Pull Request、代码评审及代表本项目参与的其他公开空间。

维护者可以编辑、隐藏或删除不当内容，关闭讨论，限制参与者访问，并在必要时向 GitHub 或相关机构报告。处理时会优先保护受影响者的隐私，并尽量说明与行为相称的处置理由。

如需报告行为问题，请使用正式部署页面公布的隐私联系邮箱联系维护者。不要在公开 Issue 中粘贴敏感证据或他人的个人信息。
